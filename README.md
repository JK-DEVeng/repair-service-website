# Repair Service Website

A real-world practice project for responsive multi-page website with backend contact & booking integration.
A responsive multi-page repair service website built using HTML, CSS, JavaScript, and Node.js.

This project simulates a real-world service business website with booking and contact functionality.

## Features
- Device-based issue selection
- Dynamic price estimation
- Clean and minimal UI
- Responsive HTML, CSS, JavaScript UI
- Dynamic price estimation
- Contact form with backend email
- Booking form with backend email integration
- Basic routing and API endpoints

## Tech Stack
**Frontend**
- HTML5
- CSS3
- JavaScript (Vanilla JS)

**Backend**
- Node.js
- Express.js
- Nodemailer (for email handling)

**Version Control**
- Git & GitHub
